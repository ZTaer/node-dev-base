# Github: node-dev-env-base 脚手架文档

<br/>
## 1. 项目目录

- node-mongodb-init

  - 适用: #开发环境#
  - node.js 动态操控 mongodb 数据库
  - 无需定义数据模型
  - 适用于快速构建操控

- node-mysql-init

  - 适用: #开发环境#
  - node.js 动态操控 mysql 数据库
  - 无需定义数据模型
  - 适用于快速构建操控

- node-mongodb-pm2-init

  - 适用: #生产环境#
  - node.js+express+mongodb+pm2
  - 可用于 node.js 后端项目生产环境开发部署
